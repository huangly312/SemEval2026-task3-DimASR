## 当前项目结构
```
SemEval2026-task3-DimABSA
    /data # 数据集
        /raw_track_a # 原始数据集
    /script # 脚本代码文件
    /subtask_1 # subtask_1相关内容（模型、config、数据处理、评估、训练、输出结果）
    ├── config.yaml              # 配置文件(数据集、模型、超参数)
    ├── main.py                  # 主程序入口（配置加载、训练/预测流程控制）
    ├── train.py                 # 训练模块（训练循环、验证、模型保存）
    ├── predict.py               # 预测模块（在测试集上生成预测结果）
    ├── model.py                 # TransformerVARegressor模型定义
    ├── Dataset.py               # VADataset数据集类（PyTorch Dataset）
    ├── utils.py                 # 工具函数集合（随机种子、实验目录、数据加载等）
    ├── evaluation.py            # 评估指标计算
    ├── example_data_processing.py  # 单个样本经过 jsonl_to_df 和 VADataset 的处理过程
    ├── experiments/              # 实验目录（按模型/语言/领域/参数组织）
    ├── logs/                    # 预测模式日志保存目录
    └── README.md                # subtask_1的说明文档
    README.md 
    修改记录.md
```
